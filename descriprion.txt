Документация по использованию ChatGPT-3.5-turbo в Telegram боте
Введение
Данный бот использует модель GPT-3.5-turbo от OpenAI для генерации ответов на сообщения пользователей в Telegram. GPT-3.5-turbo - это мощная модель языка, способная генерировать качественные, контекстно-зависимые ответы.

Функциональность
Бот поддерживает следующие команды:

/start - начинает работу с ботом.
/context_on - включает поддержание контекста. В этом режиме бот будет помнить предыдущие сообщения в беседе и использовать их для формирования ответов, создавая более естественный диалог.
/context_off - выключает поддержание контекста. В этом режиме каждое сообщение обрабатывается независимо от предыдущих.
/context_reset - сбрасывает текущий контекст. Это полезно, если вы хотите начать новый диалог с чистого листа.
Установка и запуск бота
Для установки и запуска бота необходимо выполнить следующие шаги:

Склонировать репозиторий с кодом бота.
Установить необходимые зависимости, указанные в файле requirements.txt.
Задать API ключи для OpenAI и Telegram в файле config.py.
Запустить файл bot.py.
Использование бота
После запуска бота, пользователи могут взаимодействовать с ним, отправляя сообщения в чат. В ответ на каждое сообщение бот будет генерировать ответ, используя модель GPT-3.5-turbo.

Поддержка
В случае проблем или вопросов, связанных с использованием бота, пользователи могут обратиться к документации OpenAI или к документации по API Telegram.
